Aide a l'utilisation

{% if user.FAisFA %}

En tant que FA de l'association, vous pouvez:

  • Visualiser la liste de chats sous votre responsabilite (vous pouvez toujours revenir a cette liste en cliquant sur votre nom en haut). La liste indique:
    • Le numero de registre (cliquer pour acceder/modifier les informations)
    • Le numero de puce (ou tatouage)
    • Le nom (cliquer pour faire apparaitre/disparaitre les commentaires)
    • La description du chat: sexe, age, couleur (les info manquantes sont en jaune)
    • une indication des visites veterinaires que le chat a eu (V=primo vaccination, 1=1er rappel, 2=2eme rappel, S=sterilisation/castration, P=identification, T=test FIV/Felv, R=rappel annuel, X=soins)
    • l'indication de si le chat est a l'adoption ou pas
  • Mettre a jour les informations relatives a ces chats, soit pour y rajouter des elements manquants, soit pour indiquer que le chat a ete adopte ou est decede
  • Planifier ou indiquer les visites veterinaires. Cette fonctionnalite est accessible de la page du chat en question et permet d'indiquer une visite qui a eu lieu ou encore une visite que vous devez effectuer dans le futur. Toute visite planifiee peut etre transformee en effectuee simplement en changeant l'indication et en mettant a jour les informations. Vous pouvez a tout moment effacer une visite planifiee en indiquant "ANNULLEE" et en mettant a jour. Les visites effectuees sont enregistrees definitivement et ne peuvent pas etre annullees, pensez donc a bien verifier les informations avant de valider la mise a jour....
  • A tout moment vous pouvez acceder au calendrier de visites planifiees avec le bouton correspondant
  • L'historique de toutes les visites que vous avez effectuees est enregistre dans le registre des soins
  • Vous ne pouvez pas indiquer le transfert vers une autre FA, celui-ci doit etre effectuee par un responsable de l'association
{% endif %} {% if user.FAisRF %}

Operations possibles pour un responsable FA

  • En tant que responsable FA, vous avez acces aux informations des FA sous votre responsabilite: la liste de ces FA etant accessible avec le bouton correspondant
  • Lorsque vous operez sur une autre FA, la premiere ligne de la page vous indique la FA dont vous voyez les donnees
    • Vous avez acces a leur chats comme s'il s'agissait de votres, vous pouvez donc mettre a jour les informations, planifier/indiquer des visites veterinaires, etc. Vous ne pouvez pas transferer des chats entre FA, celui-ci doit etre effectuee par un responsable de l'association
  • Vous pouvez visualiser le calendrier des visites de cette FA ainsi que son registre des soins
  • Vous avez aussi acces aux les listes des chats adoptes ou decedes
{% endif %} {% if user.FAisOV %}

Operations possibles pour un superviseur

  • Visualiser les informations de toutes les FA (liste des chats, visites)
  • Visualiser la liste de tous les chats ou celle limitee aux chats marques "adoptable"
  • Visualiser les listes des chats adoptes ou decedes
  • Vous ne pouvez en aucun cas modifier les informations relatives aux chats, ni les deplacer
{% endif %} {% if user.FAisADM %}

Operations possibles un gestionnaire registre

  • Vous pouvez rajouter un chat dans le systeme, soit manuellement soit par copier-coller d'un dossier Refugilys (il y a un format specifique a respecter!)
  • Vous pouvez effectuer toutes les operations des autres roles, en ayant donc acces a toutes le informations de tous les chats en FA, avec la possibilite des les modifier
  • Vous pouvez transferer les chats entre FA, y compris "FA speciales" (adoptes/decedes/historique), donc par exemple un chat indique decede par erreur peut etre remis dans une FA
  • Lors de l'enregistrement d'une adoption ou deces, vous pouvez transferer le chat dans l'historique
  • Vous pouvez effacer definitivement du systeme un chat de l'historique: !!! ATTENTION !!! cette operation est irreversible et efface aussi toutes les informations relatives aux visites veterinaires effectuees sur ce chat, en le faisant donc disparaitre de tous les registres de soin des FA concernees!
