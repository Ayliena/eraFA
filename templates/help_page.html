Aide a l'utilisation

{% if user.FAisFA %}

En tant que FA de l'association, vous pouvez:

  • Visualiser la liste de chats sous votre responsabilité (vous pouvez toujours revenir à cette liste en cliquant sur votre nom en haut). La liste indique:
    • Le numéro de registre (cliquer pour accéder/modifier les informations)
    • Le numéro de puce (ou tatouage)
    • Le nom (cliquer pour faire apparaître/disparaître les commentaires)
    • La description du chat: sexe, âge, couleur (les info manquantes sont en jaune)
    • une indication des visites vétérinaires que le chat a eu (V=primo vaccination, 1=1er rappel, 2=2eme rappel, S=stérilisation/castration, P=identification, T=test FIV/Felv, R=rappel annuel, X=soins)
    • l'indication de si le chat est a l'adoption ou pas
  • Mettre à jour les informations relatives à ces chats, soit pour y rajouter des éléments manquants, soit pour indiquer que le chat a été adopté ou est décédé
  • Planifier ou indiquer les visites vétérinaires. Cette fonctionnalité est accessible de la page du chat en question et permet d'indiquer une visite qui a eu lieu ou encore une visite que vous devez effectuer dans le futur. Toute visite planifiée peut être transformée en effectuée simplement en changeant l'indication et en mettant à jour les informations. Vous pouvez à tout moment effacer une visite planifiée en indiquant "ANNULLEE" et en mettant à jour. Les visites effectuées sont enregistrées définitivement et ne peuvent pas être annulées, pensez donc à bien vérifier les informations avant de valider la mise à jour....
  • A tout moment vous pouvez accéder au calendrier de visites planifiées avec le bouton correspondant
  • L'historique de toutes les visites que vous avez effectuées est enregistre dans le registre des soins
  • Vous ne pouvez pas indiquer le transfert vers une autre FA, celui-ci doit être effectuée par un responsable de l'association
{% endif %} {% if user.FAisRF %}

Opérations possibles pour un responsable FA

  • En tant que responsable FA, vous avez accès aux informations des FA sous votre responsabilité: la liste de ces FA étant accessible avec le bouton correspondant
  • Lorsque vous opérez sur une autre FA, la première ligne de la page vous indique la FA dont vous voyez les données
    • Vous avez accès à leur chats comme s'il s'agissait de vôtres, vous pouvez donc mettre à jour les informations, planifier/indiquer des visites vétérinaires, etc. Vous ne pouvez pas transférer des chats entre FA, celui-ci doit être effectuée par un responsable de l'association
  • Vous pouvez visualiser le calendrier des visites de cette FA ainsi que son registre des soins
  • Vous avez aussi accès aux les listes des chats adoptés ou décédés
{% endif %} {% if user.FAisOV %}

Opérations possibles pour un superviseur

  • Visualiser les informations de toutes les FA (liste des chats, visites)
  • Visualiser la liste de tous les chats ou celle limitée aux chats marques "adoptable"
  • Visualiser les listes des chats adoptés ou décédés
  • Vous ne pouvez en aucun cas modifier les informations relatives aux chats, ni les déplacer
{% endif %} {% if user.FAisADM %}

Opérations possibles un gestionnaire registre

  • Vous pouvez rajouter un chat dans le système, soit manuellement soit par copier-coller d'un dossier Refugilys (il y a un format spécifique à respecter!)
  • Vous pouvez effectuer toutes les opérations des autres rôles, en ayant donc accès à toutes le informations de tous les chats en FA, avec la possibilité des les modifier
  • Vous pouvez transférer les chats entre FA, y compris "FA spéciales" (adoptés/décédés/historique), donc par exemple un chat indique décédé par erreur peut être remis dans une FA
  • Lors de l'enregistrement d'une adoption ou décès, vous pouvez transférer le chat dans l'historique
  • Vous pouvez effacer définitivement du système un chat de l'historique: !!! ATTENTION !!! cette opération est irréversible et efface aussi toutes les informations relatives aux visites vétérinaires effectuées sur ce chat, en le faisant donc disparaître de tous les registres de soin des FA concernées!
